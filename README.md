# Sistema de Notas em JavaScript

Projeto criado para praticar:

- Funções
- Condicionais (if/else)
- Validação de dados

## Funcionalidade

O programa calcula a média de três notas e informa se o aluno foi:

- Aprovado
- Em recuperação
- Reprovado

## Tecnologias

- JavaScript (Node.js)

## Aprendizado

Neste projeto aprendi a organizar funções e validar dados.
